Description détaillée

Replacement for PHP's native \NumberFormatter class.

The only methods currently supported in this class are:

  • __construct
  • create
  • formatCurrency
  • format
  • getAttribute
  • getErrorCode
  • getErrorMessage
  • getLocale
  • parse
  • setAttribute
