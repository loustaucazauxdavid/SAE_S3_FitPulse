static compare ( string $version1,
string $version2,
string $operator,
?int $precision = null )
static

Compares two versions with an operator.

This method is identical to version_compare(), except that you can pass the number of regarded version components in the last argument $precision.

Examples: 

Version::compare('1.2.3', '1.2.4', '==')
// => false

Version::compare('1.2.3', '1.2.4', '==', 2)
// => true
Paramètres
int | null$precisionThe number of components to compare. Pass NULL to compare the versions unchanged.
Renvoie
bool
